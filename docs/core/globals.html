The key sequence for this binding. This key sequence should consist of one or more key strokes. Key strokes consist of one or more keys held down at the same time. This should be zero or more modifier keys, and one other key. Since M2+M3+<Key> (Alt+Shift+) is reserved on MacOS X for writing special characters, such bindings are commonly undefined for platform MacOS X and redefined as M1+M3+<Key>. The rule applies on the M3+M2+<Key> sequence.
